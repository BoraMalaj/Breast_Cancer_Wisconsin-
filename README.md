# Breast Cancer Wisconsin - Random Forest Classification

## Overview
This project applies Random Forest classification on the Breast Cancer 
Wisconsin dataset from Kaggle to predict whether a tumor is Malignant 
or Benign based on cell nucleus measurements.

## Dataset
- Source: Kaggle - Breast Cancer Wisconsin
- Samples: 569 patients
- Features: 30 numeric features
- Target: diagnosis (B = Benign, M = Malignant)

## Group
Group 2 - Bora Malaj
Assigned Model: Random Forest

## Steps Covered
1. Data Loading and Basic Information
2. Data Preprocessing - encoding, train/test split
3. Random Forest Classification
4. Confusion Matrix
5. Feature Importance Analysis
6. ROC Curve and AUC Score
7. Cross Validation - 5 Fold
